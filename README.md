# 安南将棋 (Annan Shogi) 

安南将棋の純粋Python実装。

## 実装しているルール

### 安南ルール（基本ルール）

- **味方の駒が縦に隣接している場合、上の駒は直下の駒の動き方に変化する**
- 直下に味方の駒がない場合は、本来の動きをする
- 直下に敵の駒がある場合も、本来の動きをする
- **王将も安南ルールの影響を受ける**: 直下に味方の駒があれば、その駒の動きになる
### 初期配置

- 通常の将棋と同じ駒を使用（9×9盤面）
- **2筋と8筋の歩が1マス前進した状態**でゲーム開始
  - 先手: 2七→2六、8七→8六に歩を配置
  - 後手: 2三→2四、8三→8四に歩を配置

### 駒の動き

通常の将棋と同じ駒種・動きを実装:

| 駒種 | 略称 | 動き |
|------|------|------|
| 歩兵 | FU | 前方1マス |
| 香車 | KY | 前方にスライド |
| 桂馬 | KE | 前方2マス+左右1マス |
| 銀将 | GI | 前方3方向+斜め後方2方向 |
| 金将 | KI | 前方3方向+左右+後方 |
| 角行 | KA | 斜め4方向にスライド |
| 飛車 | HI | 縦横4方向にスライド |
| 王将 | OU | 全8方向1マス |

成り駒（と金・成香・成桂・成銀: 金将と同じ動き、馬: 角+縦横1マス、龍: 飛+斜め1マス）

### 成りルール

- 敵陣（相手側3段）に入る/出る/内で動く場合、成ることができる
- **成りの強制はない**: 歩・香・桂を行きどころのない段に進めた場合でも、後から直下に駒を打てば動ける可能性があるので、必ずしも成る必要はない
### 禁手

- **二歩**: 同じ筋に自分の歩を2枚置けない
- **打歩詰**: 歩を打って相手の王を詰ますことはできない
- **自玉放置**: 自分の王が王手のまま手番を終えることはできない（※行きどころのない段への駒打ちは**可能**）

### 勝敗判定

- 合法手がなくなった側の負け（詰み）
- 投了による決着

## インストール

外部ライブラリ不要。Python 3.10以上で動作します。

## 使い方

```python
from annan_shogi import Game

# ゲーム開始
game = Game()
print(game)

# 指し手を適用 (SFEN形式)
game.apply("7g7f")

# 合法手の取得
moves = game.get_legal_moves()

# SFEN出力
print(game.to_sfen())

# 1手戻す
game.undo()

# 投了
game.resign()
```

## フォルダ構造

```
annan_shogi/
├── core/           # コア層: 駒・マス・盤面・持ち駒
│   ├── color.py    # 先手/後手
│   ├── piece.py    # 駒種・成り判定
│   ├── square.py   # 9×9マス目
│   ├── move.py     # 指し手 (SFEN対応)
│   ├── board.py    # 盤面管理
│   └── stand.py    # 持ち駒管理
├── rules/          # ルール層: 安南将棋固有ロジック
│   ├── movement.py         # 駒の移動方向定義
│   ├── annan_rule.py       # ★ 安南ルール
│   ├── legal_moves.py      # 合法手生成・王手判定・禁手チェック
│   └── initial_position.py # 初期配置
├── display/        # 表示層
│   └── text_display.py     # テキスト盤面表示
├── state.py        # ゲーム状態 (盤面+持ち駒+手番)
└── game.py         # ゲーム進行・SFEN入出力
```
